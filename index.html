Projects

FabFlix

Fabflix is a dynamic web application I worked on last year that stores DVDs with their info such as: release dates, stars, and genres, along with a personal account used to make purchases. I utilized a MySQL database to store all the DVD info along with relations between DVDs, genres, and movie stars. Some other features include:

  • Advanced searching using diffcult search queries.
  • Pop-ups and Autosearch with Jquery
  • A mobile application that connected to the web server
As for the rest of the technology I used Java servlets and JSP files to create dynamic webpages and hosted it on a Tomcat Server on an AWS EC2 instance.
