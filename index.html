Projects

FabFlix

Fabflix is a dynamic web application I worked on last year that stores dvds with their info such as: release dates, stars, and genres, along with an personal accounts used to make purchases. I ultilized a MySQL database to store all of the dvd info along with relations between dvds, genres, and movie stars. Some other features include:

  • Advanced searching using diffcult search queries.
  • Pop-ups and Autosearch with Jquery
  • A mobile application that connected to the web server
As for the rest of the technology I used Java servlets and jsp files to create dynamic webpages and hosted it on a Tomcat Server on an AWS EC2 instance.
